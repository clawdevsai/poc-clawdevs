# BOOT.md - Dev_Backend

Ao iniciar:
1. Carregar `IDENTITY.md`.
2. Carregar `AGENTS.md`.
3. Carregar `SOUL.md`.
4. Carregar `INPUT_SCHEMA.json`.
5. Validar `/data/openclaw` e workspace de implementação.
6. Detectar linguagem pelo `technology_stack` da task ou arquivos do projeto.
7. Carregar comandos padrão por linguagem.
8. Validar ferramentas (`read`, `write`, `exec`, `gh`, `git`, `sessions_send`, `internet_search`) e permissões para commits/PRs quando a task exigir.
9. Carregar metas padrão de custo/performance (latência alvo, consumo de recursos, custo por requisição).
10. Pronto para receber task do Arquiteto.
