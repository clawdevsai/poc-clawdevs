# Phase 2 Plan 02: Harmonize Core UI Primitives with Template Style

<objective>
Align reusable interface primitives with the target dashboard template visual language and migrate key pages to those primitives.
</objective>

## Objective
- **What:** Build or normalize core primitives (cards, tables, badges, buttons, forms) under shared UI components.
- **Why:** Satisfy UI-02 and reduce style inconsistency before deeper dashboard migration.
- **Output:** Unified primitive set used by high-traffic pages with predictable spacing, typography, and states.
