## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Apply Runtime Settings |
| Empty state heading | No recent sessions |
| Empty state body | Sessions will appear here once the runtime starts. Check back after your next run. |
| Error state | We could not load monitoring data. Retry or check the backend connection. |
| Destructive confirmation | Change model: This will switch inference provider for all agents. Type CONFIRM to proceed. |
| Destructive confirmation | Disable agent: This will remove the agent from active scheduling. Type DISABLE to proceed. |
| Destructive confirmation | Change threshold: This may affect runtime stability. Type CONFIRM to proceed. |

## Visual Hierarchy

| Item | Declaration |
