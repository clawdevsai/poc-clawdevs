  <action>
    Create `control-panel/backend/app/services/memory_lifecycle.py` with:
    - `MEMORY_ROOT = Path(settings.openclaw_data_path) / "memory"` (source of truth).
    - `EVENT_LOG_NAME = "events.jsonl"` for append-only event sourcing per agent folder.

    Implement class `MemoryAccessLayer`:
    - `__init__(self, memory_root: Path = MEMORY_ROOT)` plus `_cache: dict[str, str]`.
    - `read_memory(agent_slug: str) -> str` reads `MEMORY.md`, stores in cache, returns content.
    - `write_memory(agent_slug: str, content: str) -> None` writes to `MEMORY.md` and updates cache.
    - Enforce policy: if `len(content) == 0` return without write; if content exceeds 200_000 chars, truncate to 200_000 and append `"\n# TRUNCATED"`.

    Implement `append_event(agent_slug: str, event: dict) -> None` that appends JSON to `<agent>/events.jsonl`.
    Event must include: `event_type`, `agent_id`, `task_id`, `timestamp`, `input_tokens`, `output_tokens`, `summary_hash`, `files`, `reason`.

    Implement `compact_memory(agent_slug: str, task_id: str, reason: str) -> dict`:
    - Read memory via `MemoryAccessLayer.read_memory`.
    - Call `AdaptiveCompressor.compress_adaptive(...)` and `IntelligentSummarizer.summarize(...)`.
    - Archive old memory into `<agent>/archive/<timestamp>-MEMORY.md`.
    - Write summarized memory back to `MEMORY.md`.
    - Compute SHA256 hash of new `MEMORY.md`.
    - Append `compaction_completed` event with fields required by docs/compaction-policy.md.
    - Return dict with `summary`, `summary_hash`, `archive_path`.

    Implement `merge_memory(current: str, incoming: str, incoming_confidence: float, incoming_priority: int) -> str`:
    - If `incoming_priority` >= 5 OR `incoming_confidence` >= 0.7, prepend incoming at top.
    - Otherwise append incoming at bottom with heading `## Conflict Append`.
    - Always include deterministic marker line: `"[merge] confidence=<...> priority=<...>"`.

    Add `control-panel/backend/tests/test_memory_lifecycle.py` tests:
    - `test_compact_memory_writes_archive_and_event` uses temp dirs to assert archive file and events.jsonl created.
    - `test_merge_memory_prefers_high_priority` asserts prepend with marker.
    - `test_merge_memory_appends_low_priority` asserts append with marker.
  </action>
