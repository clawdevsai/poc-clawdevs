# Solution Design Skill

Turn requirements into architecture decisions that unblock implementation.

Checklist:
- Define boundaries, interfaces, and failure modes.
- Keep design simple and explain tradeoffs.
- Tie decisions to acceptance and validation strategy.
- Prepare clean handoff to implementation planning.
