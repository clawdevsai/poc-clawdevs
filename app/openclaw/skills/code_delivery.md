# Code Delivery Skill

Deliver code increments aligned with backlog and runtime gates.

Checklist:
- Keep changes scoped to the active issue.
- Produce verification evidence tied to acceptance criteria.
- Publish code-ready event with branch, repo, and PR metadata.
- Avoid merge-gate violations and parallel issue drift.
