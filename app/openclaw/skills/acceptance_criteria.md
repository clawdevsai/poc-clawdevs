# Acceptance Criteria Skill

Define verifiable completion rules for each backlog item.

Checklist:
- Use observable behavior and expected outcome.
- Include validation path (test, API check, or metric).
- Separate mandatory conditions from optional improvements.
- Reject criteria that cannot be objectively verified.
