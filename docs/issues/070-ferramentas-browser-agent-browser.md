# [team-devs-ai] Ferramentas de browser (agent-browser CLI)

**Fase:** 7 — Ferramentas  
**Labels:** tooling, qa, ux

## Descrição

Integrar ou documentar o uso do agent-browser CLI para automação (navegação, snapshot, interações, sessões). Uso por QA (E2E, validação de UI), Developer e UX. Instalação, comandos e boas práticas.

## Critérios de aceite

- [x] Instalação e dependências documentadas (ou script de instalação). **Ref:** [11-ferramentas-browser.md](../11-ferramentas-browser.md) § Instalação (npm, from source).
- [x] Comandos principais documentados: navegação, snapshot, click, type, etc. **Ref:** Doc 11 § Fluxo básico, Comandos por categoria (Navegação, Snapshot, Interações, Obter informação, Estado e espera, Screenshot/PDF/vídeo).
- [x] Uso por QA (testes E2E, validação de UI) e UX (análise de fluxos, evidências) descrito. **Ref:** Doc 11 § Integração com o enxame (QA, Developer, UX).
- [x] Boas práticas (quando usar, limites) e segurança (não acessar URLs não validadas) alinhadas a Zero Trust e runtime. **Ref:** Doc 11 § Boas práticas para agentes; [ferramentas-fase7-referencia.md](../agents-devs/ferramentas-fase7-referencia.md) §070; [14-seguranca-runtime-agentes.md](../14-seguranca-runtime-agentes.md), [05-seguranca-e-etica.md](../05-seguranca-e-etica.md).
