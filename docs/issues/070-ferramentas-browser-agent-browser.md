# [team-devs-ai] Ferramentas de browser (agent-browser CLI)

**Fase:** 7 — Ferramentas  
**Labels:** tooling, qa, ux

## Descrição

Integrar ou documentar o uso do agent-browser CLI para automação (navegação, snapshot, interações, sessões). Uso por QA (E2E, validação de UI), Developer e UX. Instalação, comandos e boas práticas.

## Critérios de aceite

- [ ] Instalação e dependências documentadas (ou script de instalação).
- [ ] Comandos principais documentados: navegação, snapshot, click, type, etc.
- [ ] Uso por QA (testes E2E, validação de UI) e UX (análise de fluxos, evidências) descrito.
- [ ] Boas práticas (quando usar, limites) e segurança (não acessar URLs não validadas) alinhadas a Zero Trust e runtime.
