# [team-devs-ai] Expertise em documentação (árvore de decisão, busca, citação)

**Fase:** 6 — Habilidades transversais  
**Labels:** docs, agents

## Descrição

Agentes devem navegar por árvore de decisão, buscar e descobrir em docs/agents-devs, obter o doc correto e citar a fonte. Fluxo: identificar necessidade → buscar → ler → citar.

## Critérios de aceite

- [x] Árvore de decisão (ou índice navegável) para docs/agents-devs disponível (ex.: README, índice por tema). **Ref:** [docs/agents-devs/README.md](../agents-devs/README.md) (índice por tema + link para árvore completa em [18-expertise-documentacao.md](../18-expertise-documentacao.md) §1).
- [x] Fluxo documentado: identificar → buscar (por tema/palavra-chave) → ler doc → citar fonte na resposta. **Ref:** [agents-devs/README.md](../agents-devs/README.md) "Fluxo: identificar → buscar → ler → citar"; [18-expertise-documentacao.md](../18-expertise-documentacao.md) §4 "Fluxo de uso".
- [x] Integração com SOUL/TOOLS ou AGENTS.md: "antes de afirmar que não tem a informação, buscar na doc do projeto e citar." **Ref:** README agents-devs regra explícita; [18-expertise-documentacao.md](../18-expertise-documentacao.md) §2 "não responder 'não tenho essa informação' sem ter buscado na documentação do projeto".
