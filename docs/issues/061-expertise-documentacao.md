# [team-devs-ai] Expertise em documentação (árvore de decisão, busca, citação)

**Fase:** 6 — Habilidades transversais  
**Labels:** docs, agents

## Descrição

Agentes devem navegar por árvore de decisão, buscar e descobrir em docs/agents-devs, obter o doc correto e citar a fonte. Fluxo: identificar necessidade → buscar → ler → citar.

## Critérios de aceite

- [ ] Árvore de decisão (ou índice navegável) para docs/agents-devs disponível (ex.: README, índice por tema).
- [ ] Fluxo documentado: identificar → buscar (por tema/palavra-chave) → ler doc → citar fonte na resposta.
- [ ] Integração com SOUL/TOOLS ou AGENTS.md: "antes de afirmar que não tem a informação, buscar na doc do projeto e citar."
