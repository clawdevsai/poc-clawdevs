# [team-devs-ai] Habilidades CISO (CyberSec)

**Fase:** 2 — Segurança  
**Labels:** security, ciso, compliance

## Descrição

Implementar as habilidades CISO no Agente CyberSec: auditoria de infraestrutura (AWS/GCP, Docker/K8s, aplicação), varredura local do ambiente OpenClaw, conformidade (SOC 2, GDPR, ISO 27001, HIPAA), resposta a incidentes (playbooks, classificação, templates), avaliação de fornecedores (tiers, questionário, DPA). Humano-no-loop para decisões críticas.

## Critérios de aceite

- [ ] Auditoria de infraestrutura: cloud, containers, aplicação (checklist ou script somente leitura).
- [ ] Varredura local OpenClaw: config, rede, credenciais, hardening SO, guardrails; local apenas, sem envio de dados.
- [ ] Conformidade: mapeamento para SOC 2, GDPR, ISO, HIPAA com checklist ou template.
- [ ] Resposta a incidentes: playbooks, classificação de severidade, template de pós-mortem.
- [ ] Avaliação de fornecedores: tiers, questionário, DPA, desligamento; regras críticas e escalação ao Diretor documentadas.
