# [team-devs-ai] UI/UX Pro Max (triagem, entregas estruturadas, design system)

**Fase:** 10 — Frontend e UX  
**Labels:** frontend, ux, design-system

## Descrição

Triagem (plataforma, stack, objetivo), entregas estruturadas (conceito UI, fluxo UX, design system, plano de implementação), heurísticas por produto/indústria, diretrizes de UX, padrão Master + overrides para design system, padrões de saída (tokens, estados, acessibilidade). Developer e UX.

## Critérios de aceite

- [ ] Triagem documentada: plataforma, stack, objetivo do produto antes de desenhar.
- [ ] Entregas estruturadas: conceito UI, fluxo UX, design system, plano de implementação (ordem e artefatos).
- [ ] Heurísticas por tipo de produto/indústria (ex.: e-commerce, SaaS, dashboard) ou referência a checklist.
- [ ] Design system: padrão Master + overrides documentado; tokens, estados, acessibilidade.
- [ ] Integração com 23-frontend-design e 02-agentes (Developer, UX).
